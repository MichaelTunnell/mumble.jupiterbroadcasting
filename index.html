Welcome to our Mumble Server!

The following is a "Quick Start Guide" and is the only "required" reading for this server.

How do I join the discussion?

  • Follow the Setup Guide. Server info is available via the Setup Guide.
  • Opus (audio codec) is a requirement for this server. Opus is available in Mumble 1.2.4 or higher.
  • Download and Install instructions for Mumble are available at www.mumble.info

PLEASE NOTE: Mumble's official website is mumble.info, all other sites such as mumble.com, mumble.net and others are not affiliated with the Mumble project.

What are the Rules?

We are fairly lenient with participation in the server but of course there are a few rules.

If you think you may have broken a rule, or about to, stop and ask a staff member for clarification.

Our list of rules is not very exhaustive so ignorance of the rules is not an excuse. Bearing that in mind, staff members deal with rule violations on a case by case basis in as equitable a way as they can.

Who are the Staff?

The list of current staff members is available from the link on the main menu.

PLEASE NOTE: The people that run the shows and show hosts are not necessarily staff members of this mumble server.

Is there anything else I should know?

Yes, though these kind of things aren't rules but rather more like server etiquette or an code of conduct. Please, read these as well to be a member of the community.
